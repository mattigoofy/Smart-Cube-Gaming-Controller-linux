## Wikipedia XML
This folder contains the wikipedia dataset, which is not uploaded to GitHub because it is > 1 GB in size.

Get the dataset from [Simplified English](https://dumps.wikimedia.org/simplewiki/) (about 1,5 GB uncompressed), or from [English](https://dumps.wikimedia.org/enwiki/) (about 100 GB (!))
The dataset to download looks something like `simplewiki-20210401-pages-articles-multistream.xml.bz2`.
